## 定义

可学习自适应掩码（Learnable Adaptive Mask, LAM）是一种根据动态环境条件（如车速、周围危险程度）自动调整视觉注意力分布的感知机制。通过学习在不同驾驶场景下应该关注哪些区域，LAM 能够提升感知效率和决策质量，同时减少无关信息的干扰。

## 关键信息

### IRL-DAL 中的完整公式链

LAM 是极简设计，仅含 **2 个可学习参数**：θ_LAM = {α_speed, α_lidar}。

#### 输入归一化
- 速度归一化：v_norm = clamp(v_t / v_max, 0, 1)
- 危险水平：h_t = clamp((d_safe - d_min,t) / d_safe, 0, 1)
  - d_min,t = min(min(l_t), d_max)，即 LiDAR 180 维读数的最小值截断

#### 调制因子
- f_speed = 1 + α_speed · v_norm
- f_hazard = 1 + α_lidar · h_t

#### 动态强度
- w_lower = w_base,lower · f_speed · f_hazard
  - w_base,lower = 1.0，w_base,upper = 0.0

#### 垂直梯度掩码
对图像每行 y ∈ [0, H-1]：
- M_t(y) = w_base,upper + (w_lower - w_base,upper) · y / (H-1)
- 归一化：M̂_t = M_t / (max(M_t) + ε)，ε = 10^-6

#### 4 通道输入
- I'_t = concat(I_t / 255.0, M̂_t) ∈ R^{H×W×4}
- 处理流程：RGB 图像 → 叠加 LAM 掩码 → 共享 CNN 骨干 → Actor/Critic 头

#### 训练方式
- 与策略网络一起通过 **BC 端到端训练**
- 优化器：Adam，学习率 η_BC = 3×10^-4
- 梯度裁剪：max norm G_max = 0.5
- 学习率调度：plateau 检测（patience=5）
- L2 正则化选择性应用于策略权重

### 核心洞察

LAM 不是移动几何注意力中心，而是**调制空间梯度强度**：
- **高速时**：f_speed 增大，增强整个可行驶区域特征提取，扩大可用视觉范围
- **接近障碍物时**：f_hazard 放大掩码强度，提供高对比度近邻信号，精确避撞
- **物理先验**：垂直梯度模拟人类驾驶员"近处清晰、远处模糊"的注意力特性

### 消融验证

在 IRL-DAL 消融实验中，添加 LAM + SAEC 带来：
- 奖励 +16%（155.1 → 180.7）
- 碰撞率 -67%（0.15 → 0.05/千步）
- 成功率 +4.3%（92.0% → 96.3%）
- 动作相似度 +5.5%（80.2% → 85.7%）

### 核心思想

传统自动驾驶感知系统通常使用固定视野或均匀采样，导致计算资源浪费在无关区域。LAM 通过以下方式实现自适应感知：

1. **动态注意力调整**
   - 根据车速调整感知范围（高速时需要更远的前视距离）
   - 根据周围危险程度调整关注区域（危险附近需要更高分辨率）

2. **可学习性**
   - 掩码参数通过端到端训练自动学习
   - 从专家驾驶数据中学到"何时该看哪里"

3. **与规划耦合**
   - 注意力分布受下游规划任务影响
   - 规划需要的信息获得更多感知资源

### 应用场景

- 高速巡航: 关注远距离前车车道
- 城市路口: 关注交叉方向的行人和车辆
- 拥堵场景: 关注近距离跟车距离
- 夜间行驶: 关注车灯和反光标识

### 优势

- 提升感知效率: 减少不必要的计算
- 增强安全性: 关键时刻聚焦危险区域
- 类人驾驶: 模拟人类驾驶员的视觉注意力机制
